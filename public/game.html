Правила игры

Цель игры

Добраться первым от старта до финиша и набрать как можно больше навыков.

Игровое поле

Поле состоит из 36 квадратов (6×6), по которым петляет река — единый маршрут из 90 станций. Старт — первая станция, финиш — последняя.

На станциях могут встречаться: пороги (препятствия), палатки (ночлег), рыбалка и грибы (еда), магазины.

Ход игрока

Обычный ход. После броска красного кубика формируется запас хода:

Запас = кубик + 1 (весло) + силы (по желанию)

Ресурсы тратятся строго по порядку: кубик → весло → силы. Движение останавливается при встрече с порогом.

Днёвка. Вместо обычного хода: +5 сил немедленно, затем ночёвка. Можно пользоваться магазином, рыбачить, собирать грибы.

Пороги

При встрече с порогом — три варианта:

  • Остановиться — остаток хода сгорает. Следующий ход начинается с этого же порога.
  • Обнести — потратить силы (−2 / −3 / −4 при сложности ≤5 / 6–8 / ≥9) и продолжить ход.
  • Пройти своим ходом:
    • Сложность ≤ навыки → успех автоматически
    • Сложность = навыки + 1 → успех + +1 навык
    • Сложность > навыки + 1 → бросок синего кубика: навыки + кубик ≥ сложность → успех + +1 навык; иначе — крушение

Крушение: −3 силы (жилет отменяет), ход заканчивается, фишка остаётся у порога.

Ночной отдых

В конце каждого хода восстанавливаются силы:

УсловиеВосстановление
Всегда+1 сила
Поел+2 силы
Есть место в палатке (или палатка на лодку)+2 силы

Способы поесть (пробуются по порядку, до первого успеха):

  1. Потратить 1 единицу еды из запасов
  2. Удочка + значок рыбки → синий кубик (4–6 успех)
  3. Корзинка + значок гриба → синий кубик (4–6 успех)

Каждый способ доступен только один раз за ход.

Магазин

На клетках с магазином во время ночёвки можно купить еду: 1 монета = 1 единица.

Победа

Когда кто-то достигает финиша, текущий круг доигрывается до конца. Побеждает тот, кто финишировал с наибольшим количеством навыков. При равенстве навыков — у кого больше монет.

Покупки перед походом

У вас 25 монет. Снаряжение и навыки можно приобрести только до начала игры. Еду можно докупать в магазинах по ходу игры.
